Fract-ol

Multithreaded fractal exploration program written in C

Usage

 git clone https://github.com/sebamiro/fract-ol && cd fract-ol
 make
 ./bin/fract-ol

Parameter list:
- Mandelbrot
- Julia
- Burning ship
- Mandelbar
- Heart
- Buffalo
- Perpendicular mandelbrot
- Celtic mandelbrot
- Celtic mandelbar
- Perpendicular celtic
